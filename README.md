# Лабораторна робота №1
## Крос-платформне програмування

### Тема
Java Reflection API

### Виконав
Падафет Олексій, група КБ22

## Опис проєкту

У проєкті реалізовано 5 завдань з використанням Java Reflection API:

1. Аналіз структури класу
2. Аналіз стану об’єкта
3. Динамічний виклик методів
4. Робота з масивами та матрицями
5. Динамічні proxy-об’єкти

## Структура проєкту

- Task1 – аналіз класу
- Task2 – аналіз об’єкта
- Task3 – виклик методів
- Task4 – робота з масивами
- Task5 – proxy та profiling

## Технології

- Java
- Reflection API
- Swing
- Proxy API
